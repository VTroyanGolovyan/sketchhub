Инструменты

1) Карандаш (p) - предназначен для рисования линий.

2) Рука (h) - предназначена для пролистывания документа.

3) Прямоугольник (s) - предназначен для рисования прямоугольников(shift - квадрат).

4) Треугольник - предназначен для рисования треугольников.

5) Эллипс (j) - предназначен для рисования эллипсов(shift - круг).

6) Прямая (b) - предназначен для рисования прямых линий.

7) Слой (l) - предназначен для создания слоев.

8) Изображение (i) - предназначен для вставки картинок(каждая картинка = новый слой).

9) Ластик (g) - предназначен для стирания нарисованых элементов.

10) Пипетка (n) - предназначена для получения цвета точки.

11) Заливка (f) - предназначена заливки области.

12) Перемещение (m) - предназначено для смещения слоев.

13) Растяжка (w) - предназначено для изменения размеров слоя.

14) Ножницы (u) - предназначены для обрезки слоев.

15) Текст (q) - предназначен для рисования текста.

Слои

Справа находиться панель слоев, в ней находяться миниатюры слоев их названия и кнопки управления. Активный слой подсвечивается ,белым контуром на панели слоев Cлой можно Удалить(клавиша delete удаляет активный слой или кнопка с мусорным ведром) Для востановления последнего удаленного слоя нужно нажать ctrl+e;

Также слои можно двигать

Каждый слой имеет историю изменений для возврата назад или вперед используйте стрелки справа вверху или ctrl+z, ctrl+y

Цвета

Сверху слева находиться панель цветов. Также можно выбрать любой цвет. Для этого нужно нажать на большой квадрат цвета.
